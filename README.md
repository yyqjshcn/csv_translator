# AI CSV 翻译助手

基于产品文档实现的单页网页应用，支持：

- 本地保存模型服务商、模型 ID、API Key、Base URL
- 聊天式输入翻译指令并上传 `.csv`
- 先做意图解析，再按批次翻译 CSV 中包含中文的字段
- 显示批次进度、失败重试，并导出 `[原文件名]_translated.csv`

## 使用方式

1. 在当前目录启动一个静态服务：

```bash
python3 -m http.server 4173
```

2. 浏览器打开 `http://localhost:4173`
3. 在左侧填写模型配置和 API Key
4. 上传 CSV，在输入框输入例如：`把这份表格翻译成英语、日语和韩语`

## 已实现的架构

- Step 1：调用轻量请求解析目标语言
- Step 2：按批次调用模型翻译 JSON 数组
- Step 3：合并结果并导出 CSV
- 异常处理：未配置密钥、缺少文件、无中文列、CSV 解析失败、接口失败重试

## 注意事项

- 这是纯前端版本，API Key 只会保存在浏览器本地存储中。
- 浏览器直连不同模型厂商接口时，是否允许跨域取决于对方接口策略；如果你在企业网关或代理下使用，可以把 `API Base URL` 改成兼容地址。
- OpenAI Compatible 模式适合多数兼容 `/v1/chat/completions` 的网关。
